Detección de reordenamiento triple en H7N9 mediante Análisis Topológico de Datos (TDA)
Este repositorio contiene el código y los resultados del experimento realizado para detectar eventos de reordenamiento triple en el virus de influenza H7N9 utilizando Análisis Topológico de Datos (TDA).

Estructura del repositorio

secuencias: Contiene las secuencias genómicas utilizadas en formato FASTA.
programas_utilizados: Contiene los scripts de Python y en Bash para el procesamiento de datos y la aplicación de TDA.
resultados: Contiene los resultados generados, incluyendo diagramas de persistencia, código de barras y mapas de calor.
